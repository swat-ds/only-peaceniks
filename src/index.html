Currently Licensed Kanopy Titles

Browse Kanopy titles currently licensed by Swarthmore College Libraries. Why? Kanopy is not Netflix, and like many other schools, Swarthmore College Libraries adopted saner access policies in 2020. One drawback: There is no simple way to see what we currently have access to.
This tool is still under development. Questions, concerns, issues? Let us know: digitalscholarship@swarthmore.edu
